network-utfpr
Este é um projeto de cabeamento estruturado, realizado como trabalho na Universidade Tecnológica
Federal do Paraná - Campus Cornélio Procópio.

Este projeto foi realizado utilizando o Latex Studio. Existe um arquivo gerado através do Latex 
com o nome de "projeto.pdf" onde algumas imagens ficaram fora de ordem e outro arquivos
chamado de "projeto-de-cabeamento-extruturado.pdf" onde apresenta as imagens nos locais corretos.
